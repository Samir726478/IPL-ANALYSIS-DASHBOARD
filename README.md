# 🏏 IPL Analysis Dashboard

## 📌 Project Overview
An end-to-end data analysis project on IPL cricket data using SQL for data extraction and Power BI for interactive dashboard visualization.

## 🛠️ Tools Used
- **MySQL** — Data storage and querying
- **Power BI** — Interactive dashboard
- **Excel/CSV** — Data export and transformation

## 📊 Key Business Questions Answered
1. Which team won the most IPL matches?
2. Who are the top run scorers of all time?
3. Which bowlers took the most wickets?
4. Which venues hosted the most matches?
5. Does winning the toss give a match advantage?
6. Who won the most Player of the Match awards?
7. How did total runs trend across seasons?
8. Who hit the most sixes?

## 💡 Key Insights
- Mumbai Indians are the most successful IPL team with 142 wins
- Virat Kohli is the all-time top scorer with 8,286 runs
- YS Chahal is the top wicket taker
- Teams choosing to field win 54% of the time
- CH Gayle holds the record for most sixes with 354
